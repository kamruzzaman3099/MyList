# 📺 Sports Live TV — M3U Player
 
A lightweight, fully browser-based **live sports TV player** hosted on GitHub Pages. It auto-loads channels from `Sports.m3u8`, plays HLS streams, and includes an embed button so you can drop the player into any WordPress site or webpage.
 
🔗 **Live Site:** [https://kamruzzaman3099.github.io/MyList/](https://kamruzzaman3099.github.io/MyList/)
 
---
 
## ✨ Features
 
- 📋 **Auto-loads** channels from `Sports.m3u8` in the same repo
- 📺 **Channel sidebar** with logo, number, and name
- 🔍 **Live search** to filter channels instantly
- ▶️ **HLS playback** via [hls.js](https://github.com/video-dev/hls.js/) — works in all modern browsers
- 🍎 **Safari support** — uses native HLS on iPhone/iPad
- ⚠️ **Error overlay** with a Retry button if a stream fails
- 📱 **Mobile responsive** — sidebar stacks above player on small screens
- 🔗 **Embed button** (bottom-right) — generates an iframe code to paste into WordPress or any site
